### Generar iconos para Android y iOS

Agregar en el archivo pubspec.yaml las siguientes lineas:

```
dev_dependencies:
  flutter_launcher_icons: "^0.7.3"

flutter_icons:
  android: true
  ios: true
  image_path: "assets/images/icono_app.png"
  ```
